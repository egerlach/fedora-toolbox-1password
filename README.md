# My Fedora Toolbox

Officially lives at
[Codeberg](https://codeberg.org/egerlach/fedora-toolbox-1password). Mirrored at
[GitHub](https://github.com/egerlach/fedora-toolbox-1password) so that the
container can be automatically built and published in
[GHCR](https://github.com/egerlach/fedora-toolbox-1password/pkgs/container/fedora-toolbox-1password)

Based off:
[ublue-os/fedora-toolbox](https://github.com/ublue-os/toolboxes/pkgs/container/fedora-toolbox)
defined in [ublue-os/toolboxes](https://github.com/ublue-os/toolboxes).

Currently adds 1Password (poorly).
